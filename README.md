## 🧠 Knowledge Base Agent — How to Use

1. Open the link above in Chrome or Edge
2. Paste your Anthropic API key in the top bar
3. Select a mode from the sidebar

| Mode | When to use |
|---|---|
| 💬 Ask a question | EPT process rules, salary rules, Indonesia recruitment questions |
| 🎯 Match CV to JD | Paste JD and CV — get fit score, matched skills, recommendation |
| 🔍 Gap analysis | Find what is missing in a candidate profile before prescreening |
| 📋 Format JD | Paste raw JD — get clean formatted document + PDF download |
| 📞 Prescreen Script | Load a JD — get a role-specific phone call script |
